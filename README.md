## 🌟 Overview

Medical reports are often filled with complex terminology, making them difficult for patients to understand.

**MedVision AI** bridges the gap between medical data and patient understanding by using **Google Gemini AI** to analyze reports, explain findings in simple language, track symptoms, and generate professional health summaries.

Whether it's a blood test, prescription, or lab report, MedVision AI helps users gain clarity about their health within seconds.

---

## ✨ Key Features

### 📄 AI Medical Report Analysis

Upload:

* Blood Test Reports
* Prescriptions
* Medical PDFs
* Lab Reports

Gemini AI automatically:

* Extracts medical information
* Highlights abnormal values
* Summarizes findings
* Explains medical terms in simple language

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/medvision-ai.git
```

Move into the project:

```bash
cd medvision-ai
```

Install dependencies:

```bash
npm install
```

Create environment variables:

```env
GEMINI_API_KEY=YOUR_API_KEY
DATABASE_URL=YOUR_DATABASE_URL
```

Run locally:

```bash
npm run dev
```
